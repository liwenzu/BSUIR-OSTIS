По аналогии с множествами, для мультимножеств также можно сформулировать некоторые правила выполнения операций:
  1. (A ∪ B)’ = A’ ∩ B’;
  2. (A ∩ B)’ = A’ ∪ B’;
  3. (A + B)’ = A’ - B = B’ - A;
  4. (A - B)’ = A’ + B;
  5. A’ - B’ = B - A;
  6. A + B = (A ∪ B) + (A ∩ B);
  7. A \ B = (A ∪ B) – (A ∩ B) = A’ \ B’;
  8. (A - B) ∩ (B - A) = ∅;
  9. A ∪ B = (A + B) – (A ∩ B) = (A ∩ B) + A \ B = A + (B – A) = B + (A – B);
  10. A ∩ B = (A + B) – (A ∪ B) = (A ∪ B) – A \ B = A – (A – B) = B – (B – A);
  11. A – B = (A ∪ B) – B = A – (A ∩ B) = (A ∪ B) \ B = A \ (A ∩ B);
  12. A \ B = (A ∪ B) – (A ∩ B) = (A – B) + (B – A) = (A – B) ∪ (B – A) = (A + B) – 2 ⋅ (A ∩ B);
  13. A + B = (A ∪ B) + (A ∩ B) = A \ B + 2 ⋅ (A ∩ B).
