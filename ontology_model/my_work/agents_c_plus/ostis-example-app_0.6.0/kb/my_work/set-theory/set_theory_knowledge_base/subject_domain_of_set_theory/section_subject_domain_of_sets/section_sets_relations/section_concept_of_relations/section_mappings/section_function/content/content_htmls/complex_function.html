Коплексная функция - комплекснозначная функция комплексного аргумента: ƒ: C → C.
Как и комплекснозначная функция вещественной переменной может быть представлена в виде:
ƒ(z) = u(z) + iv(z), где u(z) и v(z) - вещественнозначные функции комплексного аргумента, называемые соответственно вещественной и мнимой частью функции ƒ(z).
