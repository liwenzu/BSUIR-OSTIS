Ближайшим множеством A* к нечёткому множеству А называется такое четкое множество, которому принадлежат элементы, у которых степени принадлежности множеству A больше 0.5, и не принадлежат элементы, у которых степени принадлежности A меньше 0.5. Если степень принадлежности элемента нечеткому множеству A равна 0.5, то он может принадлежать или не принадлежать данному четкому множеству.
