Пусть задано семейство множеств A(α) ⊂ X, состоящее из альфа-разрезов универсального множества X. Тогда нечеткое множество B ∈ X называется точной нижней аппроксимацией семейства множеств A(α), если оно является нижней аппроксимацией A(α) и для любой другой аппроксимации C семейства множеств A(α) выполняется включение C ⊂ B.
