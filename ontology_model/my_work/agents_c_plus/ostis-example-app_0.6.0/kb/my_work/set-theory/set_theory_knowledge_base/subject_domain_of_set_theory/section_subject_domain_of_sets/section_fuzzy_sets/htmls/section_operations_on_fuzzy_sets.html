Пусть заданы два нечетких множества A c функцией принадлежности μA и B c функцией принадлежности μB на универсуме X.
 Операции над нечёткими множествами

  • Пересечением нечётких множеств A и B называется наибольшее нечёткое подмножество, содержащееся одновременно в A и B:
    μA∩B = min(μA(x), μB(x))
  • Произведением нечётких множеств A и B называется нечёткое подмножество с функцией принадлежности μAB(x) = μA(x)μB(x).
  • Объединением нечётких множеств A и B называется наименьшее нечёткое подмножество, содержащее элементы A или B:
    μA∪B = max(μA(x), μB(x))
  • Арифметической суммой нечётких множеств A и B называется нечёткое подмножество с функцией принадлежности μA+B = μA + μB - μA(x)μB(x).
  • Отрицанием(дополнением) множества A называется множество Ā с функцией принадлежности μĀ(x) = 1 - μA(x) для каждого x ∈ X.
  • Разностью двух нечетких множеств A и B, заданных на универсуме X, называется третье множество C = A\B, заданное на этом же универсуме X, функция принадлежности которого определяется по формуле: μC(x) = max(μA(x) – μC(x), 0).
