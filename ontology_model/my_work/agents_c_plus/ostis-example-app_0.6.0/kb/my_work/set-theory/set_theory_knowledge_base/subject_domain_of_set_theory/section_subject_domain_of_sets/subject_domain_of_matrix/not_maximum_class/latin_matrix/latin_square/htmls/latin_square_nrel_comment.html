Имеется большое количество нерешенных задач относительно латинских квадратов :

  1. Оценка максимального числа трансверсалей в латинском квадрате n-го порядка
  2. Характеризация латинских подквадратов в таблице умножения луп Муфанг
  3. Оценка плотности частичного квадрата, удовлетворяющего свойству Блэкберна
  4. Вычисление максимального t(n), при котором 2t(n) делит величину L(n)
