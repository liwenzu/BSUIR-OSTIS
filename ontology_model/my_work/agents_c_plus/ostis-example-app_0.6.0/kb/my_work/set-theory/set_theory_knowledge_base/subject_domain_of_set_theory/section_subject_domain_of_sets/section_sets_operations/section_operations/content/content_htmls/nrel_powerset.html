Множество всех подмножеств множества М называется булеаном и обозначает­ся 2М:

  • 2М = {А | А ⊆ М}
Теорема. Для конечного множества
  • М |2М| = 2|M|
Доказательство. Индукция по |M|. База: если |M| = 0, то М = ∅ и 2∅ = {∅}. Следовательно,
  • |2∅| = |{∅}| = 1 = 20 = 2|∅|.
Индукционный переход: пусть ∀ M |М| < k → |2М| = 2|M|. Рассмотрим М = {а1,... ,ak}, |М| = k. Положим
  • M1 = {X ⊂ 2М | ak ∈ X} и М2 = {X ⊂ 2М | ak ∉ X}.
Имеем 2М = M1 ∪ М2 и Mi ∩ М2 = ∅. По индукционному предположению |M1| = 2k-1, |M2| = 2k-1. Следовательно, |2M| = |M1| + |M2| = 2k-1 + 2k-1 = 2 x 2k-1 = 2k = 2М.

Пример. Пусть X = {1,2,3}. Тогда множество всех подмножеств X будет
2X = {0, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, X}.
