Теорема (или парадокс) Хаусдорфа — доказываемое в теории множеств утверждение о существовании счётного подмножества T двумерной сферы S2, дополнение S-2= S2∖ T которого может быть представлено в виде объединения трёх непересекающихся множеств A , B и C, конгруэнтных друг другу и множеству B ∪ C. Впервые опубликована в 1914 году Ф. Хаусдорфом. Эта теорема (как и основанная на её идеях парадокс удвоения шара) демонстрирует несоответствие теоретико-множественных представлений обычной геометрической практике.
