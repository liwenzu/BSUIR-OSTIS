Отношение φ = <Ф, М> называется отношением неравенства, если Ф = М2 \ Δ М, т.е. (∀ a, b ∈ М) (a φ b → a ≠ b). Например, задано φ = <Ф, М>, М = {1, 2, 3}, Ф = {<1, 2>, <2, 3>, <3, 1>}. Данное отношение является отношением неравенства. Отношения «5 > 3» и «3 < 10» также являются примерами отношения неравенства.
