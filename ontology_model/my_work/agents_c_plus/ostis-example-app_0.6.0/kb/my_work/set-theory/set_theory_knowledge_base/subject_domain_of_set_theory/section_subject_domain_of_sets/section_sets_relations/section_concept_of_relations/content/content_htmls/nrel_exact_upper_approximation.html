Пусть задано семейство множеств A(α) ⊂ X, состоящее из альфа-разрезов универсального множества X. Тогда нечеткое множество B ∈ X называется точной верхней аппроксимацией семейства множеств A(α), если оно является верхней аппроксимацией A(α) и для любой другой аппроксимации C семейства множеств A(α) выполняется включение B ⊂ C.
