Performances graph in accordance with the different definitions we call different kinds of performances. Between different types of representations of the graph there is one correspondence. Indeed, because it is a representation of the graph, it can be considered a set of vertices numbered. Then the arc (edge ​​will be regarded as a pair of oppositely directed arcs), going from vertex i to vertex j corresponds to an ordered pair (i, j) or, equivalently, in the list of node i will attend the vertex j, and the matrix A = (a ij), representing a graph element a ij = 1. The absence of an arc going from vertex i to vertex j, corresponds to the absence of j at the top of the list of node i, and a ij = 0.

In view of the above correspondence between the different kinds of ideas we can use different definitions of the same concepts - graph. Thus the study of various properties of the graph each time we try to use the language that is most convenient for the selected property. Probably should not choose one of languages ​​as a single language graph theory. Sometimes the description of a property, an attribute graph requires a specific language. So if we are talking about a flat (planar) graph, we necessarily have to use geometric language of graph theory. If we are talking about the "spectrum" of the graph, we will formulate a concept on the matrix language.

Graph theory is remarkable that the challenges (problems) appear in it and easily formulated immediately after the formulation of the basic concepts of graph theory.
