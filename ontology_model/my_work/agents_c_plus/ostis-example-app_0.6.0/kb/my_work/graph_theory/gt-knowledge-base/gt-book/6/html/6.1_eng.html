Research maze

The search process on the graph becomes instructive when to present it in terms of the equivalent problem, which has a long and interesting history - the problem find a way out of the maze, which consists of the intersections, connected by corridors. This section provides a detailed analysis of the basic research method of each corridor in a given maze. For some labyrinths specific task can be solved with simple rules, but for the greater part of the labyrinth requires more complex strategy. Use of terms maze (maze) instead of graph, a corridor (passage), instead of the edge and the intersection (intersection) instead of the vertex is nothing else than a semantic difference, but at this stage of the transition to a different terminology will feel deeper problem.

We believe that at every intersection mounted lamps that are turned off in the initial state, and the doors at both ends of each corridor, which in the initial state are closed. Further, we believe that the doors are built in windows and lights are powerful enough and corridors quite straight, so we opened the door, we can determine whether or not the intersection osveshen at the other end of the corridor (even if the door at the other end of the corridor is closed).

Our goal is to light all the lights and open all the doors.

To achieve this goal, we must have at its disposal a set of rules that will be followed systematically. Next research strategy labyrinth, which we call the study of three, known at least since the nineteenth century:

(i) if the current intersection no closed doors, proceed to step (iii). Otherwise, open any door of any of the corridor leading from the intersection of the current (and leave it open).

(ii) If you see that the lamp is installed at the intersection at the other end of the corridor is already on, try to open another door at the current intersection (step (i)). Otherwise (if you see that the intersection at the other end of that corridor is not illuminated), follow the corridor to the intersection, while unwinding the thread, turn on the lights and go to step (i).

(iii) if all the doors are open at the current intersection, check to see whether you are in a starting point. If so, then the process is finished. If not, use the thread to move back along the corridor that led you to this crossroads for the first time, unwinding the thread as you go and look for another closed door is already there (ie, return to step (i)).

Property 6.1. When we conduct research Tremaux a maze, we light all the lights and open all the doors in the maze and completing the tour where it started.

Proof. To prove this assertion by induction, we first note that it is done in the trivial case, ie, for the labyrinth, which contains one intersection and no corridor - we just turn on the light. For any of the labyrinth, which contains more than one intersection, we believe that this property holds for all labyrinths with fewer intersections. It suffices to show that we have visited all the corners, as we open the doors at each visit crossroads. Now look at the first corridor, which we choose at the first intersection, and divide all crossings into two subsets: (i) those that we can achieve by this corridor, and without returning to the starting point, and (ii) those that we can not achieved without returning to the starting point. By the induction hypothesis, we know that visited all corners in (i) (ignoring all the corridors, which return to the initial intersection, which is illuminated) and returned to the initial intersection.

Therefore, applying the induction hypothesis again, we know that visited all corners (ignoring the corridors leading from the starting point at the crossroads in (ii), which are illuminated).

There are four different situations that arise in the selection of the next corridor, and that we should consider taking one of the possible solutions:

(i) The corridor is not illuminated, therefore we choose.

(Ii) The corridor has already been used (in which we unwound the thread), so we choose it (and wound yarn into a ball).

(Iii) The door at the other end of the hall is closed (but the crossroads lit), by virtue of this fact we miss this corridor.

(Iv) The door at the other end of the corridor is open (and the intersection is lit), by virtue of this fact we miss this corridor.

The first and second situation refers to any passage that we bypass, with the first one end and then the other.
