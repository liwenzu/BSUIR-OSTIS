The input of the algorithm is given to the connected connected undirected graph . For each edge, its weight is specified. First, an arbitrary vertex is taken and a edge incident to this vertex and having the smallest weight . The edge found and the two vertices it connects form the tree . Then, we consider the edges of the graph, one end of which is a vertex belonging to the tree, and the other is not; From these edges the edge of the least value is chosen. The edge selected at each step is attached to the tree. Thus, when each step of the algorithm is executed, the height of the tree is increased by 1. The tree grows until all the vertices of the original graph have been exhausted.
