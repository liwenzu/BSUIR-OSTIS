Алгоритм Флойда — Уоршелла является эффективным для расчёта всех кратчайших путей в плотных графах, когда имеет место большое количество пар рёбер между парами вершин. В случае разреженных графов рёбрами неотрицательного веса лучшим выбором считается использование алгоритма Дейкстры для каждого возможного узла.
