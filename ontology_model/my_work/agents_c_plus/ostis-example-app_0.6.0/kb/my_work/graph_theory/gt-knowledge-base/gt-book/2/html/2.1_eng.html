Types of graphs

The graph G = (V, E) is called complete graph if for every pair of vertices v i and v j in V exists an edge (vi, vj) in an undirected graph =(V,)Other words, for each pair of vertices in graph G there must exsist at least one arc that connects them (Pic. 5a).

The graph G = (V, E) is called symmetric graph if there is an opposite oriented arc (v j, vi) for every arc (v i, vj) that the set of arcs E to contains (Pic. 5b).

Antisymmetric graph is a graph, which satisfies the following condition: there is no opposite oriented arc in the set A, if the arc (vi, vj) ∈ A, (Pic. 5, c). It is obvious that there are no loops in antisymmetric graph.

Combining the definitions of complete, symmetric and antisymmetric graphs, we will get the following ones:

  • graph G = (V, E), in which any pair of vertices (v i, vj) connects with bidirectional arcs, is called a complete symmetrical graph (Pic. 5d);

  • graph G = (V, E), with the only one arc for each pair of vertices (vi, vj) is called a complete antisymmetric graph or a tournament.
